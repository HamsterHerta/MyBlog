# 性缘是否应该指女男之间的关系
## 导语
>[!note]
>对于女权主义者，女♂之间的生殖行为不应该被称为“性行为”，因此用“性缘”来指代女男间的关系是有失偏颇的。
>在“性行为”♂权定义中，“性行为”是以满足♂性欲望为目的，以♂性高潮为结束，而女性则被客体化为完成这一目的的工具。
>因此在女权的视角中，这种不以女性为中心的行为不能被称为“性行为”，更不应该将其与以女性为中心的行为混为一谈。

## 

“性缘”这个被激进女权广泛提及的词语，如果深究其含义，则会吊诡地发现——在非女权群体中“性缘”包含并主要指女男之间由交配产生的关系；而在女权群体中，这个词则不应该指女男之间。换句话说，在女权的视角下，女男之间的交配行为不应该被称为“性行为”，更那么这种关系也不应该被称为“性缘”。

在女权的议题中，“反纳入”受到了广泛的认同，其支撑性的论据是——绝大多数女性无法从女男“性行为”（主要是纳入式“性行为”）中获得快感，同时纳入式“性行为”对女性而言具有极大的风险。

这样的事实其实隐含了♂权社会对“性行为”的定义——大多数♂权“性行为”虽然有女性参与，但本质上却与女性无关，它存在的是以“生殖”和“满足♂性欲望”为目的，并以“♂性高潮”为结尾。在这个过程中，女性被客体化为满足♂性欲望的工具，而以满足女性为目的的“性行为”要么不存在，要么被降级为“前戏”沦为到达真正目的的工具。

在♂权如此定义“性行为”的现实下，无论对“性”和“性欲”抱有什么态度，一个真正的女权主义者不能也不应该沉沦于这种以完全以♂性出发的定义中，而将女男间的生殖行为囫囵地归为“性行为”，将这种关系囫囵地归为“性缘”。这关系到女性的主体性，以及是否真正以女性的视角看待世界。
